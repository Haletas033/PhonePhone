# PhonePhone

## About:
PhonePhone is a phone simulator (not an emulator) made mostly for fun.

## Made with:
* C++
* Qt

## Projected Feature:
* Ability to run and download apps
* Internet access
* Custom scripting language to prgram apps
* Multiple languages
